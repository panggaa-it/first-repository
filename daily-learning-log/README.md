Tanggal: 12 Januari 2026
Topik yang dipelajari: Pengenalan Git dan GitHub
Masalah yang ditemui: Masih bingung perbedaan Git dan GitHub
Solusi / Catatan: Git adalah tools version control di lokal, sedangkan GitHub adalah layanan penyimpanan repository online. Keduanya saling terhubung.

--------------------------------------------------

Tanggal: 13 Januari 2026
Topik yang dipelajari: Membuat repository dan clone repo
Masalah yang ditemui: Salah menempatkan folder repository di komputer
Solusi / Catatan: Memastikan direktori kerja sebelum menjalankan perintah git clone atau git init.

--------------------------------------------------

Tanggal: 14 Januari 2026
Topik yang dipelajari: Git add, commit, dan push
Masalah yang ditemui: File tidak ikut ter-push ke GitHub
Solusi / Catatan: File harus ditambahkan terlebih dahulu menggunakan git add sebelum melakukan commit.

--------------------------------------------------

Tanggal: 15 Januari 2026
Topik yang dipelajari: Mengatasi error saat git push
Masalah yang ditemui: Push ditolak dengan pesan fetch first
Solusi / Catatan: Menjalankan git pull terlebih dahulu untuk menyamakan repository lokal dengan repository di GitHub.

--------------------------------------------------

Tanggal: 16 Januari 2026
Topik yang dipelajari: Membuat README dan dokumentasi sederhana
