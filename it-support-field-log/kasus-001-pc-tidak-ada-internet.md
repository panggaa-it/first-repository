\# Kasus 001 - PC Tidak Ada Internet



\## Keluhan

User melaporkan PC tidak dapat terhubung ke internet.



\## Analisa Awal

Kemungkinan:

\- Kabel LAN terlepas

\- IP Address belum terkonfigurasi

\- Gateway salah



\## Langkah Troubleshooting

1\. Cek kabel LAN

2\. Cek IP Address menggunakan ipconfig

3\. Set ulang IP Address

4\. Ping ke gateway



\## Hasil

Internet kembali normal setelah konfigurasi IP diperbaiki.



\## Catatan

Selalu cek konfigurasi IP sebelum reinstall sistem.
